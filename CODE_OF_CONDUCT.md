# Code of Conduct

We are committed to providing a welcoming and harassment-free experience for everyone.

## Our standards
- Be respectful and inclusive.
- Assume good intent, but be clear and constructive.
- Harassment, discrimination, or abusive behavior is not tolerated.

## Enforcement
Project maintainers may remove content or restrict participation that violates this policy.

If you need to report an issue, open a GitHub Security Advisory or contact the maintainers privately.
